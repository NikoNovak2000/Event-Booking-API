# Event Booking API

This is a backend REST API for an app called Event Booking.

## Description

Backend application following REST API structure and using the Gin framework. It allows users to create, view, update and delete events (CRUD). It also supports user authentication, which consists of user sign up and user log in. Besides that, there is an authentication implementation via user signup and login, as well as JSON Web Tokens for extra verification.

## Getting Started

### Requirements

- IDE Editor, Visual Studio Code - [installation for VSC](https://code.visualstudio.com/Download)
- Go - [installation for Go](https://go.dev/doc/install)
- Gin - [installation for Gin](https://github.com/gin-gonic/gin)

### Executing program

- Run the programs in terminal with command

```
 go run .
```
